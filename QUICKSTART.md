# Быстрый старт

## Запуск сайта локально

1. Откройте файл `index.html` в браузере
2. Или запустите локальный сервер:

```bash
# Python
cd site
python -m http.server 8000

# Node.js
npx http-server site
```

Откройте: `http://localhost:8000`

## Что нужно настроить перед запуском

### 1. Контакты (обязательно)

Откройте `index.html` и найдите все упоминания:
- `+79143350675` - замените на ваш телефон
- `https://t.me/s/climate_hall` - замените на ваш телеграм-канал

### 2. Обработка форм (рекомендуется)

По умолчанию формы открывают WhatsApp. Для интеграции с CRM:

1. Откройте `script.js`
2. Найдите функцию `handleFormSubmit`
3. Раскомментируйте нужный метод из `integration-examples.js`
4. Укажите ваши ключи/токены

### 3. Аналитика (опционально)

Добавьте код Google Analytics или Яндекс.Метрики в `<head>` файла `index.html`

## Структура файлов

- `index.html` - основная страница
- `styles.css` - стили
- `script.js` - функциональность
- `integration-examples.js` - примеры интеграций
- `project/technical_specification.md` - техническое задание

## Развертывание

Просто загрузите файлы на хостинг:
- `index.html`
- `styles.css`
- `script.js`

Остальные файлы не обязательны для работы сайта.

## Поддержка

При возникновении проблем проверьте:
1. Консоль браузера (F12) на наличие ошибок
2. Правильность контактов
3. Работоспособность форм
